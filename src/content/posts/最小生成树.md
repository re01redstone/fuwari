# 什么是最小生成树?
给你一个带权无向图, 然后挑几条边, 让它成为代价(权值)最小的一颗树(无环图).

或者通俗一点, 就是给你一些地点(点), 然后还有一些你可以修的路(边), 用最小的代价把所有的地点连起来.

>[!NOTE]
>### 为什么是树?
>因为树是无环图, 如果说存在环, 那就可以把环去掉而不影响点的联通, 所以说权值最小的是一棵树.

# Kruskal算法
那很自然就可以想出:~~并不自然~~
1. 将边按权值从小到大排序
2. 遍历每条边
	1. 如果已经被连上了 - 跳过
	2. 如果没连上 - 连上, 并记录代价

可以发现, "判断边有没有被连上"其实就是"判断两个点是否属于同一集合"

同样的, "连上边"就是"合并集合"`

那又可以发现, 这玩意可以用并查集来实现啊, 所以就可以转化为:
对于一张邻接表`a[i]={u,v,w}`
1. 对w从小到大排序
2. 遍历a
	1. `if(find(a[i].u)==find(a[i].v))` skip
	2. `if(find(a[i].u)!=find(a[i].v))` `fa[find(a[i].u)]=find(a[i].v)` and `sum+=a[i].w`

# P3366 【模板】最小生成树

## 题目描述

如题，给出一个无向图，求出最小生成树，如果该图不连通，则输出 `orz`。

## 输入格式

第一行包含两个整数 $N,M$，表示该图共有 $N$ 个结点和 $M$ 条无向边。

接下来 $M$ 行每行包含三个整数 $X_i,Y_i,Z_i$，表示有一条长度为 $Z_i$ 的无向边连接结点 $X_i,Y_i$。

## 输出格式

如果该图连通，则输出一个整数表示最小生成树的各边的长度之和。如果该图不连通则输出 `orz`。

## 输入输出样例 #1

### 输入 #1

```
4 5
1 2 2
1 3 2
1 4 3
2 3 4
3 4 3
```

### 输出 #1

```
7
```


样例解释：

 ![](https://cdn.luogu.com.cn/upload/pic/2259.png) 

所以最小生成树的总边权为 $2+2+3=7$。

## 示例代码
```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
const ll N=1e6;
struct Node{
    ll u,v,w;
}a[N];// 边结构体
ll fa[N],n,m,ans=0;

bool cmp(Node a,Node b) {return a.w<b.w;}// 比较函数
ll find(ll x){// 并查集找爹函数
    if(fa[x]==x) return x;
    return fa[x]=find(fa[x]);
}

int main(){
    cin>>n>>m;
    for(int i=1;i<=n;i++) fa[i]=i;
    for(int i=1;i<=m;i++) cin>>a[i].u>>a[i].v>>a[i].w;
    sort(a+1,a+m+1,cmp);// 按照w从小到大排序
    for(int i=1;i<=m;i++){
        if(find(a[i].u)!=find(a[i].v)){// 判断是否已经连接(属于同一个集合)
            fa[find(a[i].u)]=find(a[i].v);// 连接, 属合并到一个集合
            ans+=a[i].w;// 记录代价
        }
    }
    cout<<ans<<"\n";

    return 0;
}
```
